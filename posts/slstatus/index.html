slstatus

November 27, 2022


slstatus is a status monitor for window managers that use WM_NAME or stdin to fill the status bar.

Installation

git clone https://github.com/xyzsuf/slstatus.git
cd slstatus
sudo make clean install

Go to your $HOME directory, and open “.xprofile” and paste the following:

slstatus & 
exec dwm

After you reboot, the status bar will reload.
