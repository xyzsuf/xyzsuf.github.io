Sed

November 25, 2022

What is sed?


Sed is a stream editor, which is used to manipulate text files.

We will work with a sample text file to understand how to use sed.


Sample Text File

  this is a test
  hello world 
  1 am sed  
  one two three 
  3 am coffee

Syntax

sed s/regularexpression/replace/g filename

NOTE:

Using sed without the -i does not modify the file.

To modify, use sed -i

Example 1:

Since Global variable “g” is missing in the end, it replaces only the first “t” in each line of text.

sed s/t/T/ test


Output: 

This is a test
hello world
1 am sed
one Two three
3 am coffee

Example 2:

Using the global variable “g”

sed s/t/T/g test


Output: 

This is a TesT
hello world
1 am sed
one Two Three
3 am coffee

This replaces all t’s in the text file.


Example 3:

sed s/ee/ing/g test


Output:

this is a test
hello world
1 am sed
one two thring
3 am coffing

Substitute Beginning and End of line.

Beginning

Syntax

sed s/^beginofword/replace/g

Example:

sed ’s/^t/yy/g'


Output:

yyhis is a test
hello world
1 am sed
one two three
3 am coffee

End

sed s/endofword$/replace/g

Example:

sed ’s/ee$/oooo/g'

Output:

this is a test
hello world
1 am sed
one two throooo
3 am coffoooo
