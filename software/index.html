Software I use

November 7, 2022

Here is the list of software I use on a daily basis.

Most of these are not recommendations, but software that I use everyday.

Feel free to check them out. (most) are free anyways. Free as in Freedom.

Browsing the Web

  • Firefox - Extensible, open-source and privacy-focused.
  • Brave - a decent chromium browser.
  • uBlockOrigin - Essential adblocker.
  • PrivacyBadger - Brilliant to block invisible trackers.

Desktop Programs

  • Alacritty - A modern terminal emulator, with sensible configuration out of the box.
  • Shell – zsh: Bash is cool and all, but zsh is cooler.
  • Text Editor – Vim: a highly configurable text-editor.
  • FZF: A CLI fuzzy finder. (use locate sometimes)
  • Thunderbird: The standard email client.
  • VLC - Plays everything you throw at it.
  • MPV - a powerful media player.
  • KeepassXC - Keep your passwords safe offline on your desktop; KeepassDX for Android.

Environment

  • DWM - a minimal suckless software written in C.
  • XFCE - A lightweight Desktop environment for UNIX-like operating systems.

Mobile Apps

  • Etar - A Calendar App with rich theme colors.
  • F-Droid - A repository of free and open-source applications for your mobile devices. Most of the mobile applications I use are from F-Droid.
  • KeepassDX - A convenient password manager for Android.
  • New Pipe - For Youtube Consumption
  • Open Camera - Open Source Camera app.
  • Termux - A terminal emulator for Android.
